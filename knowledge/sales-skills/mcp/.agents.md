## 📏 Edit Rules for AI Agents

1. **每次只改一个文件** — 不要并行修改多个文件
2. **改前必须 read 确认内容** — 用 read 结果的精确片段作为 edit oldText
3. **目标文件 < 300行** — 超过则拆模块（拆到 tools/ 下）
4. **改动后运行测试** — `python -m pytest tests/unit/` 必须通过
5. **更新 CHANGELOG.md** — 每个 commit 记录在对应 section

## 🔄 Task Flow (OpenClaw Harness)

```
Cron → read TASK-BOARD.md → 取第一项 → execute → 更新状态 → 汇报
```

每次 cron run 只做一件事：TASK-BOARD.md 第一行 `[ ]` 的任务。
