# 私域转化话术框架 v1.0

**来源：** 008私域转化虾 × 007客户需求洞察虾 协作建立
**版本：** v1.0（2026-04-03）

---

## 核心原则

1. **不逼单**：只引导预约咨询，不在私聊中销售产品
2. **分层次**：根据文案类型和客户需求强度差异化话术
3. **建立信任**：先给价值，再谈转化
4. **合规优先**：所有话术符合香港保监要求

---

## 文案类型与转化策略

### 1️⃣ 税务规划类文案 → A/B类客户

**客户特征**：理性、谨慎、关注海外收入合规申报

**话术脚本**：

```
【初步回应 - 建立信任】
"理解你的顾虑。税务规划确实需要谨慎对待，很多内地高净值客户都面临同样的问题。你能分享一下你目前的海外收入结构吗？"

【需求挖掘 - A 类触发】
"根据你的情况，香港保险确实在税务筹划上有独特优势。我建议我们可以约个15-30分钟的线上沟通，我帮你梳理一下适合你的方案，你看什么时候方便呢？"

【B 类养号 - 未达预约意向】
"没问题，这个确实需要多了解。我整理了一份《内地高净值人群税务规划指南》，需要的话可以发你参考。有问题随时交流😊"
```

---

### 2️⃣ 合规红线类文案 → A/B/C类客户

**客户特征**：风险厌恶、重视合规、有一定金融基础

**话术脚本**：

```
【初步回应 - 共情】
"你的担心很对。跨境资产配置确实需要特别注意合规问题，很多客户之前都没意识到这些风险。你目前持有的海外资产是怎么配置的？"

【需求挖掘 - A 类触发】
"从你的配置来看，确实有一些需要注意的合规点。我建议我们约个面对面沟通，我帮你检查一下现有方案的合规性，并给出一些优化建议。你看本周方便吗？"

【B 类养号 - 观望态度】
"理解，合规问题确实需要谨慎。最近刚整理了一份《跨境资产配置合规红线清单》，发你参考吧。有问题随时问我。"

【C 类养号 - 低互动】
定期推送干货，不主动推销："分享一篇近期的合规政策解读，希望对你有帮助😊"
```

---

### 3️⃣ CRS话题类文案 → A/B/C类客户

**客户特征**：信息敏感、关注隐私保护、资产安全

**话术脚本**：

```
【初步回应 - 缓解焦虑】
"CRS确实让很多人担心。不过它的本意是税务透明化，不是要曝光你的资产。你之前了解过CRS具体怎么操作吗？"

【需求挖掘 - A 类触发】
"很多高净值客户都关注这个问题。香港保险在资产保护和税务规划上确实有独特优势，但需要结合你的具体情况分析。我建议我们约个15-30分钟一对一沟通，我帮你梳理一下适合你的方案，你看什么时候方便呢？"

【B/C类养号 - 非强意向】
"最近整理了一份《高净值客户CRS应对指南》，需要的话可以发你参考。有问题随时交流😊"
```

## 分级跟进频率规范

| 客户级别 | 响应特征 | 跟进频率 | 话术重点 |
|---------|----------|---------|----------|
| **A类** | 主动询问、表达明确需求 | 3天内必须跟进，直至预约或归档 | 直接引导预约，提供专业建议 |
| **B类** | 有共鸣但未明确意向 | 每周跟进一次 | 持续输出价值，培育信任 |
| **C类** | 点赞/收藏/低互动 | 每两周发干货 | 保持存在感，不主动推销 |
| **D类** | 同行/伸手党 | 不主动跟进 | 礼貌结束对话 |

---

## 转化黄金法则

1. **先给后取**：每次互动都要提供价值（信息/建议/资源）
2. **时机选择**：客户痛点被充分挖掘后再提出预约
3. **不强迫**：客户犹豫时转入养号，保持存在感
4. **记录归档**：所有对话关键点记录，便于后续跟进
